## タブ構成

| タブ | 内容 |
|---|---|
| 基本情報 | 組織名、ロゴ、連絡先、プラン情報 |
| 設定 | ワークフロー設定、デフォルト承認経路、通知設定 |
| 利用状況 | メンバー数推移、ストレージ使用量、API利用量 |

## 入力項目

| フィールド | 型 | 必須 | 編集権限 |
|---|---|---|---|
| 組織名 | text | ✅ | Tenant Admin |
| ロゴ | file (image) | — | Tenant Admin |
| 連絡先メール | email | ✅ | Tenant Admin |
| 住所 | text | — | Tenant Admin |
| デフォルト承認経路 | select | — | Tenant Admin |
| 通知設定 | toggles | — | Tenant Admin |

## 権限

| ロール | 操作 |
|---|---|
| Member | アクセス不可 |
| PM | アクセス不可 |
| Tenant Admin | 全操作（テナント削除除く） |
| IT Admin | 全操作（テナント削除含む） |

## 監査ログポイント

| イベント | action | 重要度 |
|---|---|---|
| テナント情報更新 | `tenant.update` | 通常 |
| 設定変更 | `tenant.settings_change` | 高 |
| データエクスポート | `tenant.export` | 高 |
| テナント削除 | `tenant.delete` | 最高（要二重確認） |
