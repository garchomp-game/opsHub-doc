## 目的 / In-Out / Related
- **目的**: ユーザーが参加中のプロジェクトを一覧・検索・管理する
- **対象範囲（In/Out）**: 一覧・フィルタ・新規作成導線。詳細は SCR-C01-2
- **Related**: REQ-C01 / SPEC-API-C01 / DD-DB-projects

## 画面情報
- **画面ID**: SPEC-SCR-C01-1
- **画面名**: プロジェクト一覧
- **対象ロール**: 全ロール（表示範囲がロールにより異なる）
- **URL**: `/projects`
- **状態**: Draft

## UI構成（概要）
- **テーブル列**: プロジェクト名 / ステータス / PM / 期間 / メンバー数 / 進捗
- **フィルタ**: ステータス（計画中/進行中/完了/中止）、PM
- **ソート**: 更新日（デフォルト降順）
- **「新規作成」ボタン**: PM / Tenant Admin のみ表示
- **ページネーション**: 20件/ページ

## 振る舞い・遷移
- 行クリック → `/projects/[id]`（SCR-C01-2）へ遷移
- 「新規作成」→ 作成モーダル or `/projects/new`
- Member: 自分がアサインされたプロジェクトのみ表示（RLS）
- PM: 自分が管理するプロジェクトを表示

## 監査ログポイント
- 一覧表示自体はログ対象外
