## タブ構成

| タブ | 内容 | データソース |
|---|---|---|
| 概要 | 基本情報、進捗、メンバー、アクティビティ | `projects`, `tasks`, `timesheets` |
| タスク | タスク一覧（→ SCR-C02 の簡易ビュー） | `tasks WHERE project_id = :id` |
| 工数 | メンバー別・月別の工数集計テーブル | `timesheets WHERE project_id = :id` |
| メンバー | PJ所属メンバー一覧と役割 | `project_members` |
| 経費 | PJに紐づく経費一覧 | `expenses WHERE project_id = :id` |

## 入力項目（編集モード）

| フィールド | 型 | 必須 | 編集権限 |
|---|---|---|---|
| プロジェクト名 | text | ✅ | PM / Tenant Admin |
| 顧客名 | text | — | PM / Tenant Admin |
| ステータス | select | ✅ | PM / Tenant Admin |
| 期間（開始/終了） | date range | ✅ | PM / Tenant Admin |
| 予算 | number | — | PM / Accounting |
| 説明 | textarea | — | PM |

## 権限

| ロール | 操作 |
|---|---|
| Member | 閲覧（自分が所属するPJのみ） |
| PM | 編集・メンバー追加/削除 |
| Accounting | 閲覧 + 予算・経費タブ |
| Tenant Admin | 全PJ閲覧・編集 |

## 監査ログポイント

| イベント | action |
|---|---|
| PJ情報更新 | `project.update` |
| メンバー追加 | `project.add_member` |
| メンバー削除 | `project.remove_member` |
| ステータス変更 | `project.status_change` |
